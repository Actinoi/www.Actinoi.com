题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

  1. 院士奖学金，每人元，期末平均成绩高于 分 ，并且在本学期内发表篇或篇以上论文的学生均可获得；
  2. 五四奖学金，每人元，期末平均成绩高于分，并且班级评议成绩高于分的学生均可获得；
  3. 成绩优秀奖，每人元，期末平均成绩高于分的学生均可获得；
  4. 西部奖学金，每人元，期末平均成绩高于分的西部省份学生均可获得；
  5. 班级贡献奖，每人元，班级评议成绩高于分的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是分，班级评议成绩分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。

输入输出格式

输入格式：

第一行是个整数N(1≤N≤100)，表示学生的总数。

接下来的N行每行是一位学生的数据，从左向右依次是姓名，期末平均成绩，班级评议成绩，是否是学生干部，是否是西部省份学生，以及发表的论文数。姓名是由大小写英文字母组成的长度不超过的字符串（不含空格）；期末平均成绩和班级评议成绩都是到之间的整数（包括和）；是否是学生干部和是否是西部省份学生分别用个字符表示，Y表示是，N表示不是；发表的论文数是到的整数（包括和）。每两个相邻数据项之间用一个空格分隔。

输出格式：

包括行。

第行是获得最多奖金的学生的姓名。

第行是这名学生获得的奖金总数。如果有两位或两位以上的学生获得的奖金最多，输出他们之中在输入文件中出现最早的学生的姓名。

第行是这N个学生获得的奖学金的总数。

输入输出样例

输入样例#1：

4
YaoLin 87 82 Y N 0
ChenRuiyi 88 78 N Y 1
LiXin 92 88 N N 0
ZhangQin 83 87 Y N 1

输出样例#1：

ChenRuiyi
9000
28700

说明

2005提高组第一题

题解：

​ 这道题目直接模拟就可以通过，注释也比较多，就不详细讲解了。

#include <cstdio>
#include <cstring>
#include <iostream>
using namespace std;
char cadre[101], western[101]; //存储姓名，是否是学生干部，是否是西部省份学生
string name[101];
int fa[101], ca[101], thesis[101], num[101], sum[101], ans = 0; //存储期末平均成绩，班级评议成绩，发表的论文数量，每个学生姓名的长度，每个学生的奖金总数以及这N个学生获得的奖学金的总数
void judge (int i) { //判断能得到多少奖金
    if ((fa[i] > 80) && (thesis[i] >= 1))
        sum[i] += 8000; //院士奖学金，每人8000元，期末平均成绩高于80分（>80），并且在本学期内发表1篇或1篇以上论文的学生均可获得
    if ((fa[i] > 85) && (ca[i] >80))
        sum[i] += 4000; //五四奖学金，每人4000元，期末平均成绩高于85分（>85），并且班级评议成绩高于80分（>80）的学生均可获得
    if (fa[i] > 90)
        sum[i] += 2000; //成绩优秀奖，每人2000元，期末平均成绩高于90分（>90）的学生均可获得
    if ((fa[i] > 85) && (western[i] == 'Y'))
        sum[i] += 1000; //西部奖学金，每人1000元，期末平均成绩高于85分（>85）的西部省份学生均可获得
    if ((ca[i] > 80) && (cadre[i] == 'Y'))
        sum[i] += 850; //班级贡献奖，每人850元，班级评议成绩高于80分（>80）的学生干部均可获得
    ans += sum[i];
    return;
}
int main(){
    int n; //表示学生的总数
    cin >> n;
    int maxx = 0, identifier = 0; //存储获得最多奖金的学生的钱数以及编号
    for (int i = 1; i <= n; i++) {
        cin >> name[i] >> fa[i] >> ca[i] >> cadre[i] >> western[i] >> thesis[i];//学生姓名，期末平均成绩，班级评议成绩，是否是学生干部，是否是西部省份学生，以及发表的论文数
        judge(i);
        if (sum[i] > maxx){
            maxx = sum[i];
            identifier = i; //更新信息
        }
    }
    cout << name[identifier] << endl; //获得最多奖金的学生的姓名
    cout << maxx <<  endl; //输出这名学生获得的奖金总数
    cout << ans; //输出这N个学生获得的奖学金的总数
    return 0;
}
